# SQL Database Project
This project demonstrates my foundational SQL skills through the design and creation of a relational database from scratch.

## Project Overview
The goal of this project is to practice database design, table creation, relationships, and basic SQL queries in a structured and organized way.

## Skills Used
- SQL
- Database design
- Table creation
- Primary and foreign keys
- Data insertion
- Basic queries

## Project Structure
- `schema.sql` – database creation
- `tables.sql` – table definitions
- `insert_data.sql` – sample data insertion
- `queries.sql` – example queries for exploring the data

## Purpose
This project was created as part of my learning journey in data analytics and SQL, helping me build a strong foundation in relational database thinking.
